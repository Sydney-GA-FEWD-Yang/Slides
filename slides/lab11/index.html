Programming task

Enter a mark and determine if it's a

  • HD - Mark >= 85
  • DN - Mark >= 75
  • CR - Mark >= 65
  • PS - Mark >= 50
  • FL - Mark < 50

Variables

First get the mark from the user and assign it to a variable


            var mark = $('#mark').val();

Now the conditionals


            var result; //Unassigned at this point
            if (mark < 50) {
                result = 'FL';
            } else if (mark < 65) {
                result = 'PS';
            } else if (mark < 75) {
                result = 'CR';
            } else if (mark < 85) {
                result = 'DN';
            } else {
                result = 'HD';
            }
            $('#result') = result; //Print result to screen
