Terminal Commands

Change directory 

            cd mydirectory/folder
Check version 

            git --version
node --version
npm --version

Git Cheatsheet

Clone 

            git clone https://github.com/user/repo.git foldername
Commit a change 

            git add .
git commit -m "your log message"
Push to the cloud 

            git push origin master
Pull from the cloud (if you're working on multiple computers 

            git fetch origin master
