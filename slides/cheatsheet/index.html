Lab00 Cheatsheet

Git and version control

Terminal Commands

Change directory 

            cd mydirectory/folder
Check version 

            git --version
node --version
npm --version

Git Cheatsheet

Clone 

            git clone https://github.com/user/repo.git foldername
Commit a change 

            git add .
git commit -m "your log message"
Push to the cloud 

            git push origin master
Pull from the cloud (if you're working on multiple computers 

            git fetch origin master

Lab01 Cheatsheet

HTML Basics

Anchor Tag (Links)


            <a href="http://www.google.com">Google</a>

Open in new tab


            <a href="http://www.google.com target="_blank">Google</a>

Link tag

Used to link external stylesheets

CSS format


            TagName {
    target: attribute;
}

Example


            h2 {
    color: red;
}
