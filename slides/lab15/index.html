Forms

An important part of web development is forms. Without forms, you can't collect data from users.

Simple Login

Lets look at each individual attribute

Collecting the data


$('#login').submit(function(event){
    event.preventDefault();
    //Your code
})

Example


$('#login').submit(function(event){
    event.preventDefault();
    var username = $('#username').val();
    var password = $('#password').val();
    if (username === 'yang' && password === 'qwerty'){
        alert('logged in!');
    } else {
        alert('wrong login');
    }
})

preventDefault

  • Why do we need event.preventDefault?
  • Default behaviour of a form submit is to refresh the page
  • Modern websites are dynamic
  • Modern websites aims to change the content without refreshing

Ajax!

Ajax revolutionized web development

Before ajax

Traditional websites would refresh to load new data

For example - http://www.w3schools.com/

What is ajax?

Asynchronous Javascript and XML

XML is also old - replaced with JSON

JSON - Javascript Object Notation

Ajax allows us to get data without refreshing

Ajax with jQuery

There are two main types of AJAX request

  • GET
  • POST

There are others, but you can get away with just these two.

These requests are part of what's called Restful services

GET Request

Used to get data from a server


$.get('http://url.com', function(data, status){
    //Do what you need
});

data is normally in Object notation

POST Request

Used to send the server data


$.post('http://url.com', data, function(data, status){
    //do what you need
})

Note here data is the data you send to the server

Data is in Object notation
