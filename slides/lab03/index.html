CSS Box Model

Using CSS


            div{
    margin: 10px 20px 30px 40px;
    border: 5px solid black;
    padding: 40px 30px 20px 10px;
}

Margin

In the above, we used 4 different pixels in margin. These represents top right bottom left


            div{
    margin: top right bottom left;
}

But margin can also accept less than 4 inputs

Search for margin in google

Targetting specific margin side


            div{
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 30px;
    margin-left: 40px;
}

This is equivalent to the previous margin statement

Border

Border accepts up to 3 inputs, border-width, border-style, border-color


            p{
    border: 2px dotted blue;
}

This is equivalent to


            p{
    border-width: 2px;
    border-style: dotted;
    border-colour: blue;
}

Did you spot the mistake?

How do I know all the border-styles?

Padding

Syntax is the same as margin, just replace margin with padding


            div{
    padding: 10px 20px 30px 40px;
}

Equivalent to


            div{
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 30px;
    padding-left: 40px;
}
