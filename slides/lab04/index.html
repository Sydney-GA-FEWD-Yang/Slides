More specificity

Contrived Example:


            

    Lets select this element

In stylesheet.css


            .red-background-div{
    color: red;
}

Is this good enough?

We can select all divs containing a certain class


            div.red-background-div{
    color: red;
}

Note how there is no space between div and .red-background-div - this is intentional

Question: Order here is important, why?


            //switch position
.red-background-divdiv{
    color: red;
}

This will think you are selecting an element with the "red-background-divdiv" class

Multiple classes

On the same element


            

    Lets select this element

Question: Notice this element has two classes, what are they?

image-holder and red-background-div

How do we select this?


            .image-holder.red-background-div{
    color: red;
    border: 1px solid black;
}

Question: Note order is not important here, why?

No, Order is only important when you have a tag selector

Question: What happens if I put a space between the two classes in CSS?

It becomes a ancestor/descendant selector

Question: Can we have multiple ids?

Answer: No

Remember, all selectors follow the ancestor/descendant selector rules
