Lesson 0 - GitHub and version control

General Assembly

Git WAT?

Git is a version control system

All modern development teams use some sort of version control system (Git, SVN, Mercurial). Git is extremely popular amongst front-end web developers.

Git vs Github

Git is the name of the open source software

Github is the name of a hosting company that uses the Git software

You can think of Github and Git as the same thing for now

Benefits

  • Git allows multiple users to work on the same codebase (project) and helps track changes.
  • Github allows you to store shared code on a cloud server
  • Git allows you to revert changes, review changes before they are commited and everything else that a version control system includes

We'll be using Github throughout this course

Fun Fact: Github employees work on an open allocation basis, meaning they can work on any project they like.
