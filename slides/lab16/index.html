How to Access Objects

There are two ways

myDetails.name
myDetails['name']

The second way is more dynamic and works for everything

Lets try


var user = {
    username: 'jones',
    firstname: 'Jack',
    surname: 'Jones',
    phoneNumber: '012345678'
}

console.log(user.firstname);
console.log(user['phoneNumber']);

user.firstname = 'Amanda';

var a = 'firstname';

console.log(user[a]);
