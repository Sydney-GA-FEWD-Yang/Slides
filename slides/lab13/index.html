EM size


p {
    font-size: 1em;
}
  • em sizes are relative to the parent font-size
  • Usually used for font sizes

Default

The default size of a page is 16px

We can change this default using


body {
    font-size: 16px;
}

In this case

  • 1em = 16px
  • 2em = 32px
  • 1.5em = ?px

The case for ems

Why ems?

Ems let us change one font-size and all the font-sizes on the screen will change

Useful for Responsive designs?

Yes!

We can easily control the font-size of the page through one change

In real life

It's not used as much as it should be in real life.

Why? Here are some reasons

  • People are used to px
  • Ems isn't absolute and hence isn't easy to reason about

Media Queries

Media queries allows us to give different css to an element based on some properties

Properties may include

  • Whether the screen is a print screen
  • The width of your browser window
  • Heres a list

Syntax


@media screen and (min-width:600px) {
    body{
        color: blue;
    }
}
@media screen and (max-width:599px) {
    body{
        color: blue;
    }
}

This media query is only an example...

Real life uses


@media screen and (min-width:600px) {
    .hide-for-mobile{
        display: block;
    }
}
@media screen and (max-width:599px) {
    .hide-for-mobile{
        display: none;
    }
}

Now the hide-for-mobile class will hide content when screen is less than 599px
