Grid system!

  • Most responsive libraries will have a grid system
  • If you don't use any of the responsive widgets, grid is still very useful
  • Grid helps us escape the need for a table

12 columns

  • Grid generally works by using a 12 column system
  • Most content use the same sized columns
  • To get 3 columns, you will need to use 4 grid columns for each column
  • To get 2 columns, you will need to use 6 grid columns for each column

Content
  • Notice the row class, this is for one row of columns
  • Notice it's col-md-6, col stands for column, md stands for medium and 6 stands for 6 standard columns
  • In this case, we wanted 2 columns since 2x6 = 12

Sizes

  • col-xs-6, md stands for medium - which means it will break grid at extra small size
  • col-sm-6, md stands for medium - which means it will break grid at small size
  • col-md-6, md stands for medium - which means it will break grid at medium device size
  • col-lg-6, md stands for medium - which means it will break grid at large size
