Variables

Variables help us maintain state within our application 

            var nameOfVariable = 0;

Two things to note here

  • Use camelCase (first letter small, every other new word starts with a capital letter)
  • initialize value (in this case 0) [this is optional]

camelCase

Start your variables with a lower case letter or _ (underscore)

  • evenOdd
  • playerName
  • articleTitle

Becareful not to name your variables after keywords


            var var = 'hello'; //Wrong!
var if = 'cool'; //Wrong!

Why? What happens if we don't follow this rule?

And, Or

You can combine conditions

  • E.g, determine if person needs to work
  • Age between 18 and 65
  • Use the and && operator

var age = 17

if (age > 18 && age < 65) {
    alert('needs to work');
} else {
    alert('go play');
}

Determine if fruit

  • If you get an apple or orange then it's a fruit
  • Everything else is not a fruit
  • Use the or || operator

            var fruit = 'cabbage';

if (fruit === 'apple' || fruit === 'orange') {
    alert('fruit!');
}
