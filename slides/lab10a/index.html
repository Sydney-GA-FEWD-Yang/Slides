Arrays

Arrays represents comma delimited data

For example a list of groceries


var emptyArray = [];
var filledArray = ['apple', 'oranges', 'pork'];
var fibonacciArray = [1, 1, 2, 3, 5, 8, 13];

Arrays - Selecting Items


var names = ['yang', 'jack', 'joey', 'miranda'];
var secondName = names[1];
var firstName = names[0];
var thirdName = names(3); //Wrong!

Remember Arrays start from 0, so to select the 54th item, we use 53.

Arrays - Methods

You can search for the full list online, but heres a few.


var fruits = ['apple', 'banana', 'mango'];
fruits.push('blueberry'); //adds a fruit to the end;
fruits.pop(); //Removes the last fruit
console.log(fruits.length); //prints out how many fruits there are
console.log(fruits); //What will this print

Objects

Used to store key-value pairs


var obj = {
    name: 'Yang Li',
    occupation: 'Developer',
    location: 'Sydney'
}

Objects help you store a table of data

Objects extended

You can mix and match objects and arrays and any other data type


var customer = {
    gender: 'male',
    paid: false,
    order: [
        {
            name: 'steak',
            detail: 'medium rare with mash',
            served: false,
            price: 15.99
        },
        {
            name: 'Chicken Burger',
            detail: 'No Onions',
            served: true,
            price: 12.50
        }
    ],
    tips: 0
}

For loop


for (variable name and start value; condition to break loop; incrementer) {
    body of loop
}

Using For loops


var letters = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i' ,'j', 'k'];
for (var i = 0; i < letters.length; i++) {
    alert(letters[i]);
}

Bug!


for (var i = 0; i < 10; i--) {
    console.log(i);
}

Whats wrong with this loop?

It's an infinite loop
